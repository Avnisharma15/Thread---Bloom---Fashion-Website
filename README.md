# Thread & Bloom – Fashion Website

This is a modern and responsive fashion e-commerce website made using **HTML, CSS, and JavaScript**.  
It includes sections like Hero Slider, Collections, New Arrivals, Top Picks, Brand Partners, Newsletter, Instagram Feed, and Contact.

## Features
- Premium and stylish fashion UI  
- Fully responsive layout  
- Hero image slider (Swiper.js)  
- New arrivals carousel  
- Top-selling products slider  
- Category collections  
- Video modal section  
- Newsletter subscribe section  
- Instagram gallery  
- Smooth scroll and AOS animations  

## Technologies Used
- HTML  
- CSS  
- JavaScript  
- Swiper.js  
- AOS Animation Library  
- Font Awesome Icons  

## How to Use
1. Download or clone the project  
2. Open `index.html` in your browser  
3. All media files are inside the `Images` folder
